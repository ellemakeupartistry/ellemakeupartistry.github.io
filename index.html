How To Set Up Your Free Website

1

Create a GitHub Account

If you don't have one yet, sign up at GitHub.com.

2

Create a New Repository

Name it username.github.io (replace "username" with your GitHub username).

3

Upload This Website

Download this HTML file and upload it to your repository as index.html.

4

Enable GitHub Pages

Go to Settings > Pages and select the main branch as your source.

5

Visit Your Website

Your site will be live at https://username.github.io within minutes.
